好吧，雖然我覺得我應該不會用到，還是記錄一下好了。

  • 1. 先停止mysql deamon。
sudo mysqld stop
  • 2. 使用safe mode啟動mysql。
mysqld_safe --skip-grant-tables
  • 3. 以root方式連線至mysql。
mysql --user=root mysql
  • 4. 重設root的密碼。
update user set Password=PASSWORD('new-password') where user='root';
flush privileges;
exit;
  • 5. 重啟mysql deamon，你就可以用新密碼嘍。
