問題

在開發npm package的時候，如果想要在程式碼裡測試一下package，但又不想為此build一個暫時的package。

解法

這個時候就可以用下面的方式直接require你要的檔案：

const mycode = require('/User/your/local/package/folder/code')
// ...
mycode.bar()
