很簡單，可以用private_class_method :new

class A
  private_class_method :new
end

private_class_method也可以用在其它的class method，不過必須放在method之後才會work。

class A
  def self.my_method
    # ...
  end
  private_class_method :my_method
end

另外，如果你只是為了做singleton而做private initializer，那其實可以考慮使用ruby內建的Singleton class，不用自己重建輪子。
