問題

如果有一個時間字串，例如："2014-09-18 09:48:05"，要怎麼轉成ruby中的time或datetime object？

解法

如果需要保留時區資訊，可以使用Time.parse，回傳的是帶有時區的Time object：

require "time"
Time.parse("2014-09-18 09:48:05")
# => 2014-09-18 09:48:05 +0800

如果不需要時區資訊，可以用DateTime.parse，回傳的是不帶時區的Datetime object，這個時候如果用.to_time轉成Time object的時候，就會加入時區的計算：

require "date"
dt = DateTime.parse("2014-09-18 09:48:05")
# => #
dt.to_time
# => 2014-09-18 17:48:05 +0800

如果只是想取得日期資訊，可以用Date.parse，它只會回傳date object：

require "date"
Date.parse("2014-09-18 09:48:05")
# => #
